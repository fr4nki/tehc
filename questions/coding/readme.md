# Общие задачи

### Дана строка s которая содержит только символы '(', ')', '{', '}', '[' и ']', определить является ли строка валидной.

Строка валидна если:
- Открытые скобки должны закрываться скобками того же типа.
- Открытые скобки должны закрываться в правильном порядке.

Input: s = "()"
Output: true

Input: s = "()[]{}"
Output: true

Input: s = "(]"
Output: false

Input: s = "([)]"
Output: false

Input: s = "{[]}"
Output: true
